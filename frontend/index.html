3-Tier-Architecture

GitHub Repo AWS | Terraform | GitHub Actions

Apache Web Application hosted on an EC2 server via Terraform

Consists of a VPC covering 2 AZs (Availability Zones). Each AZ has 1 Public and 2 Private Subnets. Public Subnet contains a NAT Gateway and an EC2 instance hosting the Apache Web Server. Private Subnet contains an EC2 Instance that accesses internet through SSH via NAT Gateway. Also contains an RDS Instance that stores data, as well as an ALB (Application Load Balancer) and an ASG (Auto Scaling Group) for scaling and load distribution. Setup a GitHub Actions workflow to automate any future changes to the infrastructure.
